TSNPeerBluetooth
================
Bluetooth LE peer-to-peer library for iOS.

Using TSNPeerBluetooth
----------------------
Add TSNPeerBluetooth to your podfile.
```
pod 'TSNPeerBluetooth'
```
And install it using:
```
pod install
```
